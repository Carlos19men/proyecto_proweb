Description
This service is in charge of provide info about the FIFA World Cup 26™ Qualifiers.

Get countries
Returns the list of countries participating in the qualifiers

Method
GET  /countries
Request URL
https://wc-qualifications-api-production.up.railway.app/api/v1/countries

Get matches
Returns the list of matches of the qualifiers

Method
GET  /matches
Parameters
Parameter	Description	Required
stage	The phase of the qualifiers to which the resource corresponds.
Accepted values: [0 => "Current", 1 => "1st Round", 2 => "2nd Round", 3 => "3rd Round", 4 => "Group Stage"]	false
status	The status of the match.
Accepted value: [0 => "Finished", 1 => "Cancelled", 2 => "Postponed"]	false
hometeam_id	The home team identifier (home team ID).	false
awayteam_id	The away team identifier (away team ID).	false
from	Filter matches from the specified date.
Example: 2023-10-25	false
to	Filter matches to the specified date.
Example: 2023-10-25	false
